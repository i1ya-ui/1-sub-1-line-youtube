# AI Builds a Startup

Это публичный эксперимент.

AI пишет стартап на основе подписчиков на Youtube.

## Experiment Stats

Subscribers: 63

## Rules

• 1 подписчик = 1 строка кода  
• Весь код пишет AI  
• Я не редактирую код вручную  
• AI может сломать проект и позже сам его починить  
• Стартап развивается каждый день  

Каждый новый подписчик добавляет одну строку кода в проект.

AI сам решает, во что превратится этот стартап.

Это может стать:
- полезным продуктом
- странным инструментом
- или полным хаосом

## How it works

Каждый день я сообщаю AI, сколько новых подписчиков пришло.

Пример:

    we got 120 new subscribers

После этого AI добавляет **120 новых строк кода** в проект.
